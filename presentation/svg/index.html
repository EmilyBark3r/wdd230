SVG = Scalable Vector Graphics

SVG differs from raster images like .png and .jpg that use a grid of tiny pixels to create an image. When you zoom in on a raster image the pixels become larger making the image look grainy. A vector image can be scaled to any size without losing its resolution. This is because vector images use geometry.

Webp image format

WEBP images can compress much smaller than JPG or PNG and will speed up loading times, but are not fully supported by all browsers, so you need fallback images.
